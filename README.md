# jdiven 🌤️

[🇨🇿 Česká verze níže](#česká-verze)

> Where should you go outside? Jdiven will tell you.

A web app showing current conditions for outdoor activities in Prague. At a glance, you can see where it's best to be outside right now — based on weather, air quality, and more.

## What it does

- 🗺️ Map of Prague with a heatmap of conditions by neighbourhood
- 🌡️ Current temperature, precipitation, wind and sunshine
- 💨 Air quality index
- 📅 Outdoor events nearby *(coming soon)*
- 🚌 Public transport status *(coming soon)*

## Data sources

- [ČHMÚ opendata](https://opendata.chmi.cz) — meteorological stations
- [OpenWeatherMap](https://openweathermap.org) — current weather and forecast
- [Netatmo](https://dev.netatmo.com) — community stations across Prague
- [Prague open data](https://opendata.praha.eu) — events and city data

## Tech stack

- React
- Leaflet + react-leaflet
- Leaflet.heat

## Run locally

```bash
git clone https://github.com/tvoje-jmeno/jdiven.git
cd jdiven
npm install
npm run dev
```
